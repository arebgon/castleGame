## Prerequisites

##### Install XCode 12
* You can download XCode in [HERE](https://developer.apple.com/xcode/)

##### Get project:

```
git clone https://github.com/arebgon/castleGame.git -b main --single-branch
```

* Open castleGame.xcworkspace file
